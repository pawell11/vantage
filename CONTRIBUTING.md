# Contributing

This is a marketing site for the VANTAGE platform. Contributions related to content accuracy, design improvements, and documentation are welcome.

## Getting started

1. Fork the repository
2. Make your changes (HTML/CSS/JS only — no build step)
3. Test locally: `python3 -m http.server 8080`
4. Submit a pull request

## Guidelines

- Keep it consistent with the existing dark theme
- No third-party dependencies (no npm, no CDN scripts)
- Test that both `index.html` and `docs.html` render correctly
- Ensure responsive layout works on mobile viewports

## Code of conduct

Be respectful. This project serves a sensitive domain — keep content professional.
