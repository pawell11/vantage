# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.x     | ✅ Active          |
| < 4.0   | ❌ Not supported   |

## Reporting a Vulnerability

Report vulnerabilities to **vantageaiservices@gmail.com**. Do not open public issues for security concerns.

You can expect:
- **Acknowledgment** within 48 hours
- **Status update** within 5 business days
- **Fix timeline** communicated once root cause is understood

## Encryption

Reported vulnerabilities are encrypted in transit via TLS 1.3. This is a static marketing site — no user data is collected or stored.
